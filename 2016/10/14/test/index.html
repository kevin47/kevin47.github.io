  1. Modern Ethernet networks built with switches no longer need CSMA/CD because each Ethernet segment is isolated. I.e. changed from bus network (broadcast their frames) to star network (send their frames to specific MAC address).
  2. a)

        Hidden terminal problem or hidden node problem occurs when AB, BC are visible to each other, but AC is not. Since AC can’t see each other, they may send packets to B simultaneously, and collisions occur.

       

        Exposed terminal problem or exposed node problem occurs when AB, BC, CD are visible to each other. B’s sending packets to A prevents C from sending packet to D, because carrier sense will conclude that it will interfere with the transmission by its neighbor B.

       

        b)

        To overcome the hidden term problem, RTS/CTS uses handshake, i.e. the sender first ask if it can send packets to the receiver, and only continues if the receiver say yes. Since the receiver wouldn’t say yes to multiple senders, the hidden terminal problem is avoided.

        RTS/CLS deals with the exposed terminal problem as follows: When a node hears an RTS but not the corresponding CTS, that node can deduce that it wouldn’t be interfering with others’ transmission and is permitted to transmit to other neighboring nodes.
